# Zib OS Logo Spec (v0)

## Core symbol set

The branding uses a 4-glyph set: `• ♥ ☺ ~`

## Canonical 2x2 layout

Use this fixed order in all primary lockups:

```text
•  ♥
☺  ~
```

## Spacing and geometry

- Use a square container.
- Keep equal cell sizes for the 2x2 grid.
- Horizontal and vertical gaps should be equal.
- Leave outer padding around the grid equal to one cell gap.
- Keep optical centering (slight nudge allowed if one glyph appears visually heavy).

## Color guidance

- Primary dark background: `#1E1E2E`
- Foreground (light): `#CDD6F4`
- Accent option: `#89B4FA`

## Usage

- Launcher icon: centered 2x2 mark.
- Boot splash: symbol mark plus `zib OS` wordmark.
- CLI banner: ASCII version from `branding/logo-ascii.txt`.
